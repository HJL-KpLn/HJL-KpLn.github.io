有一个坑是，文章里提到的theme next，已经不更新了，在后续对数学公式的支持很差。主要开发者换了一个地方，从https://github.com/next-theme/hexo-theme-next这个仓库安装比较好。后续基本按照上面的blog进行即可

我使用的数学公式支持是hexo-renderer-markdown-it-plus

功能实现

图片直接用md语法索引

参考官方文档，无法直接实现用md语法索引图片，而是需要用{% asset_img example.jpg This is an example image %}来实现。或者安装hexo-renderer-marked实现。

后者的问题在于，安装后无法正常显示数学公式。所以改换其他方法。

搜到这篇解决方法，即：

  1. 安装hexo-image-link，npm i -s hexo-asset-link
  2. 修改hexo/_config.yml：post_asset_folder: true

然后索引图片时，就可以直接使用md语法了，如：
![alt text](./your_blog/your_image)

侧边栏的目录和分类等

NEXT theme里面自带的，可以在themes/next/_config.yml里面设置，在menu下。

需要注意的是，menu里的about, schedule, sitemap, commonweal这四个选项，需要自己设置网页

设置的方法（以about为例）：到根目录，hexo new page about，然后编辑about/index.html即可
