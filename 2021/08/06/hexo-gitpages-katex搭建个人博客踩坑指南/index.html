功能实现

图片直接用md语法索引

参考官方文档，无法直接实现用md语法索引图片，而是需要用{% asset_img example.jpg This is an example image %}来实现。或者安装hexo-renderer-marked实现。

后者的问题在于，安装后无法正常显示数学公式。所以改换其他方法。

搜到这篇解决方法，即：

  1. 安装hexo-image-link，npm i -s hexo-asset-link
  2. 修改hexo/_config.yml：post_asset_folder: true

然后索引图片时，就可以直接使用md语法了，如：
![alt text](./your_blog/your_image)
